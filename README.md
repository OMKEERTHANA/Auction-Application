# BidMaster - Online Auction Platform

BidMaster is a full-stack web application built using React, Node.js, Express, and MongoDB. It allows users to register, create auctions, and place bids in real time. Admins can manage users, auctions, and bids through a dedicated dashboard.

## Features
- User authentication (Login & Register)
- Create and manage auctions
- Real-time bidding system
- Admin dashboard
- Bid history tracking

## Tech Stack
- React (Frontend)
- Node.js & Express (Backend)
- MongoDB Atlas (Database)

## How to Run
1. Clone the repo  
2. Run `npm install`  
3. Add `.env` file  
4. Run backend: `node server.js`  
5. Run frontend: `npm run dev`
